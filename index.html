CSS Cheat Sheet

CSS Shortcut

CSS Code Description
color Changes the color of the text
background-color Changes the color of the background of the selector
text-shadow Allows user to apply shadow to text
border-color Single color to use for every side of the element's border

HTML Shortcut

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of rows containing actual table data
